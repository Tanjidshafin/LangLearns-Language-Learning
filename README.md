## 🎯 Overview  

**Boighor** is a **Library Management System (LMS)** designed to simplify **library operations** in schools and institutions. It provides an efficient way to manage books, borrowing, and returning while ensuring a seamless **user experience for students, teachers, and administrators**.  

📌 **Why Use Boighor?**  
✔️ **Efficient & Organized** book tracking system.  
✔️ **User-friendly interface** for smooth interaction.  
✔️ **Secure authentication** for protected transactions.  
✔️ **Responsive design** ensuring accessibility on all devices.

## ✨ Features  

- **📖 Book Management**: Easily **add, update, and categorize** books.  
- **🔄 Borrow & Return System**: Track borrowed books with **due dates**.  
- **🔍 Search & Filter**: Quickly find books by **title, author, or category**.  
- **🌓 Dark Mode**: Switch between **light & dark themes** for a better reading experience.  
- **🔑 Role-Based Access**: **Admins manage books**, while users can **borrow & track their books**.  
- **📱 Fully Responsive**: Works smoothly across **mobile, tablet, and desktop**.

## 🚀 Getting Started  

### Prerequisites  

Ensure you have the following installed:  
- [Node.js](https://nodejs.org/)  
- [Git](https://git-scm.com/)  

### Installation Steps  

1️⃣ **Clone the repository**  
```bash
git clone https://github.com/yourusername/boighor.git
cd boighor
```

2️⃣ **Install dependencies**  
```bash
# Install frontend dependencies
cd client
npm install

# Install backend dependencies
cd ../server
npm install
```

3️⃣ **Set up environment variables**  
Create a `.env` file in the **server** directory and add:  
```env
MONGO_URI=your_mongodb_connection_string
FIREBASE_API_KEY=your_firebase_api_key
```

4️⃣ **Start the project**  
```bash
# Start the backend server
cd server
npm start

# Start the frontend development server
cd ../client
npm start
```

🔗 The project should now be running at **http://localhost:3000/** 🚀
