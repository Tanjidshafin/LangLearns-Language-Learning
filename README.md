## 🚀 Quick Start Guide

<details>
<summary>Click to expand</summary>

### 📋 Prerequisites

- [Node.js](https://nodejs.org/)
- [Git](https://git-scm.com/)

### 🔧 Installation

1️⃣ **Clone the magic**
```bash
git clone https://github.com/yourusername/langlearns.git
cd langlearns
```

2️⃣ **Summon the dependencies**

```shellscript
npm install
```

3️⃣ **Ignite the project**

```shellscript
npm start
```

4️⃣  Visit [http://localhost:3000/](http://localhost:3000/) and start learning! 🎉

</details>
